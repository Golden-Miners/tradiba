# Tradiba

Tradiba is an offline-first AI-powered trading research platform built on top of Hermes Agent.

## Status

- Version: v0.1.0 (Foundation - In Progress)
- Markets: XAUUSD, XAGUSD
- Data Source: MetaTrader 5
- Execution: Manual only

The repository will be built incrementally with production-quality code, tests, and documentation.
